Properties

avatar string null

Sets custom avatar at the chip's left corner. The avatar may be represented by image(if the value is an url to image), plain text or HTML.

Code examples

Set the avatar property. 

 <smart-chip avatar='A'></smart-chip>

Set the avatar property by using the HTML Element's instance. 

 const chip = document.querySelector('smart-chip');
 chip.avatar = '../avatar.jpg';

Get the avatar property. 

 const chip = document.querySelector('smart-chip');
 let avatar = chip.avatar;
closeButton boolean false

Sets or gets whether close button is displayed.

Code examples

Set the closeButton property. 

 <smart-chip close-button></smart-chip>

Set the closeButton property by using the HTML Element's instance. 

 const chip = document.querySelector('smart-chip');
 chip.closeButton = false;

Get the closeButton property. 

 const chip = document.querySelector('smart-chip');
 let closeButton = chip.closeButton;
disabled boolean false

If disabled is set to true, the interaction with the element is denied.

Code examples

Set the disabled property.

Set the disabled property by using the HTML Element's instance.

 chip.disabled = false;

Get the disabled property. 

 const chip = document.querySelector('smart-chip');
 let disabled = chip.disabled;

Events

close Event

This event is triggered when the chip is closed.

Code examples

Bind to the close event by type: smartChip 

 const chip = document.querySelector('smart-chip'); 
 chip.addEventListener('close', function (event) {
     // event handling code goes here.
 }

Methods

close Method

Closes the chip.

Parameter Type Description
None
Return Value
None

Code examples

Invoke the close method.

 chip.close();
