Properties

appendTo any null

Specifies the container where new openned alert items will be displayed. The value can be an HTMLElement or element's id. This property is in relation with modal(lower priority than modal) and position(higher priority than position) properties.

Code examples

Set the appendTo property. 

 <smart-alert append-to='container'></smart-alert>

Set the appendTo property by using the HTML Element's instance. 

 let alert = document.querySelector('smart-alert');
 alert.appendTo = newContainer;

Get the appendTo property. 

 let alert = document.querySelector('smart-alert');
 let appendTo = alert.appendTo;
autoClose boolean false

Sets or gets whether the alert will automatically close after duration equal to the autoCloseDelay property.

Code examples

Set the autoClose property. 

 <smart-alert auto-close></smart-alert>

Set the autoClose property by using the HTML Element's instance. 

 let alert = document.querySelector('smart-alert');
 alert.autoClose = false;

Get the autoClose property. 

 let alert = document.querySelector('smart-alert');
 let autoClose = alert.autoClose;
autoCloseDelay number 3000

Sets or gets the duration after which the alert automatically closes (works only if the autoClose property is set to true).

Code examples

Set the autoCloseDelay property. 

 <smart-alert auto-close-delay='4000'></smart-alert>

Set the autoCloseDelay property by using the HTML Element's instance. 

 let alert = document.querySelector('smart-alert');
 alert.autoCloseDelay = 5000;

Get the autoCloseDelay property. 

 let alert = document.querySelector('smart-alert');
 let autoCloseDelay = alert.autoCloseDelay;

modal boolean false

In modal mode the alert item is positioned in the center of the screen. This property is with higher priority than position and appendTo. If modal is set to true these properties are disregarded.

Code examples

Set the modal property. 

 <smart-alert modal></smart-alert>

Set the modal property by using the HTML Element's instance.

Get the modal property.

position string top-right

Sets the part of the browser window where the alert will be positioned. The position property is disregarded if appendTo or modal are set. Possible values: 'top-left', 'top-right', 'bottom-left', 'bottom-right'.

Code examples

Set the position property.

Set the position property by using the HTML Element's instance.

Methods

closeAll Method

Closes all opened alert items.

Parameter Type Description
None
Return Value
None

Code examples

Invoke the closeAll method. 

 let alert = document.querySelector('smart-alert'); 
 alert.closeAll();
closeItem Method

Closes particular alert item.

Parameter Type Description
item HTMLElement/String The alert item (or its id) to remove.
Return Value
None

Code examples

Invoke the closeItem method. 

 let alert = document.querySelector('smart-alert'); 
 alert.closeItem(item5);
