Properties

currentItem number 0

Gets/sets current item's number.

Code examples

Set the currentItem property. 

 <smart-pager current-item='5'></smart-pager>

Set the currentItem property by using the HTML Element's instance. 

 let pager = document.querySelector('smart-pager');
 pager.currentItem = 10;

Get the currentItem property. 

 let pager = document.querySelector('smart-pager');
 let currentItem = pager.currentItem;
