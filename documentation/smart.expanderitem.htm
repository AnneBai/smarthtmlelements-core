Properties

arrow string left

Sets or gets header's arrow position. If the value is 'none' the arrow is not shown. Possible values: 'left', 'right', 'none'.

Code examples

Set the arrow property. 

 <smart-expander-item arrow='right'></smart-expander-item>

Set the arrow property by using the HTML Element's instance. 

 let expanderItem = document.querySelector('smart-expander-item');
 expanderItem.arrow = 'none';

Get the arrow property. 

 let expanderItem = document.querySelector('smart-expander-item');
 let arrow = expanderItem.arrow;
